This project predicts house prices using a Machine Learning model (RandomForestRegressor) trained on the California Housing Dataset. A Streamlit web interface is used to allow users to interact with the model and generate predictions.

To run or set up this project, follow the steps below:
 1. First, run main.py to generate model.pkl and pipeline.pkl.
 2. Then run main.py again to evaluate the model using input.csv.
 3. The script will generate an output.csv file in which the last column contains the model's predictions for each record.
 4. After that, run app.py to get the user interface for interaction, where you can enter inputs to obtain predictions based on the trained model.
